
这是一种将原始演示与加速演示进行联合训练的混合学习的通用框架。实验表明，该框架在多任务与多基线设置下实现约 1.7x的加速（仅讨论了2x数据集的情况），并保持甚至提升任务成功率。具体而言，我们设计了两个模块：第一，最优传输过滤模块（Optimal-Transport Filtering，OTF），通过最优传输轨迹距离对下采样后的演示进行无监督质量评估，过滤掉分布偏移明显、可能伤害训练的异常轨迹\cite{peyre2019computationalot,cuturi2013sinkhorn}；进一步地，我们采用一种简单的混合策略，以同时注入“精度监督”和“效率监督”。然而，我们观察到直接（naively）混合训练常出现优化不稳定与权衡漂移；为缓解该问题，我们提出用自适应的权重去平衡不同监督的权重，并且受多目标优化的启发，设计了nash权重平衡模块（Nash Weight Balancing，NWB），利用 Nash-Pareto 求解得到自适应权重与折中更新方向\cite{nash1950bargaining,sener2018mtlmo}，从而稳定地学习到更接近速度–精度 Pareto 前沿的解。
